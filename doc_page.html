Data Types

There are 5 primitive data types:

  • Float
  • String
  • Boolean
  • None Type
  • Integer

Here are some examples of each data type:

3 #Integer
 4.0 #Float
 "Hello World" #String
 True #boolean
 None #None Type

There are 4 non-primitive data types:

  • Lists
  • Dictionaries
  • Sets
  • None Type

Here are some examples of each data type:

[1,2,3,4,5] #List
 { 'England': 'London', 'Scotland': 'Edinburgh', 'France': 'Paris'} #Dictionary
 {'England', 'Scotland','France'} #Set
 (1,2,3,4,5) #Tuple

The non-primitive data types allow you to collect and modify data you may want later.

Literals

This represents any notation that represents a fixed value in the code. A literal describes a value of a data type. For example:

5 # 5 is the integer literal

Data Structures

A data structure is a datatype that contains the data in a specific way and provides means to manipulate it. For example:

country_capital = {'England': 'London', 'Scotland': 'Edinburgh', 'France': 'Paris'}
# We access dictionary value by passing its key in the square brackets:
country_capital['France']
Operations

When we do something to data we call these operations. For example adding things together:

3 + 5 # The + is the addition operator that tells python to add two numbers together
Statements

A statement is a line of code or multiple lines of code that expresses an action for the computer to perform. Every program consists of a sequence of statements. For example:

print(3+4)

The code snippet below, prints out the integer 5, this statement means that Python is performing an action.
